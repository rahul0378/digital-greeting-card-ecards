== Description ==
Create Ecard site by Wordpress and Digital-greeting-cards-ecards plugin.. 

At a glance, this plugin adds the following:

- Create Card Category.
- Create Cards on Wp admin.
- Display Cards listing on Frontend.
- Sender can send ecard to Recover Email.
- Check Sent Cards on backend.

After Active There are Auto Dummy Data Creater Link use this for create Dummy Data on your Plugin...

== Frequently Asked Questions ==


=  What to do after install and Active? =

When activated this plugin will ask for setup demo data if you not want to setup demo data go to cetegorys and create some and go to ecards and create some..

= how to setup Ecard listing page =

Go to plugin Settings => and write requred fields.

=  What to Create Ecard? =

Go to Wordpress Admin you will get menu item "All Cards->Add Card" for add card.

= How to check Sended cards? =

Go to Wordpress Admin you will get menu item "All Cards->Entries" for view sended card.
